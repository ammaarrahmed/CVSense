# Data Directory

This directory contains the dataset for the CVSense project.

## Structure

- `resumes/` - Store resume files here (PDF or text format)
- `job_descriptions/` - Store job description files here (PDF or text format)

## Data Format

### Resumes
- Format: PDF or TXT
- Naming convention: `resume_<id>.pdf` or `resume_<id>.txt`
- Content: Candidate resume with experience, skills, education, etc.

### Job Descriptions
- Format: PDF or TXT
- Naming convention: `job_<id>.pdf` or `job_<id>.txt`
- Content: Job requirements, responsibilities, qualifications, etc.

## Notes

- Ensure all files are properly formatted before processing
- Remove any sensitive personal information from resumes
- Maintain consistent naming conventions
